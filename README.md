## 📖 About

ScreenTranslator is a convenient Windows application that lets you translate any part of your screen — inside any application window. Simply select the area you need, and the app automatically detects the text there and shows its translation. It works regardless of whether the text is in a game, browser, document, or any other application window.

## ✨ Features

- 🖱️ Select Any Screen Area
- 🔍 Automatic Text Recognition (OCR)
- 🌍 Instant Translation
- 🎮 Works in Games, Browsers, Documents & More
- ⚡ Fast & Lightweight
- 🎨 Modern User Interface
- 🖥️ Windows Optimized
